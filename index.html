This file is for github repo practice

  1. First click the New Repo button
  2. Name the repo, make it public, have a readme initialized
  3. Copy the link to the repo to your clipboard
  4. Open bash and navigate to a folder where you want the repo downloaded
  5. Type git clone https://github.com/MikeWarshaw/ClassPractice
  6. Navigate into the new folder named after your repo
  7. Make or change a file ie. touch index.html
  8. From inside your repo folder type: git add -A (this tells git you have something to add
  9. Type git commit -m "a comment about the file should be between these quotes"
  10. Type: git push
  11. Check github to see that your file got uploaded
